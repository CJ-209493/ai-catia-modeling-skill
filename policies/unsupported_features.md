# Unsupported or Narrow v1.0 Boundaries

These features are not user-mode native successes in v1.0 unless a future recipe and verifier promote them:

- solid rect-to-circle multi-section loft
- full 3D rounded-path pipe sweep
- solid helical spring sweep
- sheet metal workflows
- BRep-to-BRep assembly constraints
- robust arbitrary edge/face group selection
- solid modeled thread geometry

Verified but narrow:

- Hybrid surface loft marker
- Native Thread feature, not solid helical thread geometry
- Product mono-element Fix constraint, not full coax/contact assembly constraint
